# Day2_VideoTag
This HTML page is a basic web tutorial and roadmap for web development. It features YouTube videos and locally hosted videos, along with some CSS styling for better presentation.
Embedded YouTube Videos: The page includes three embedded YouTube videos covering various web development topics.

Local Videos: Three locally hosted video elements are provided for different web development categories (frontend, full-stack, and backend).

Responsive Design: The page uses CSS to ensure responsive design for both embedded and local videos.

Styled Headings: The headings are styled with CSS to enhance the visual appeal of the page.
